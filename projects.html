Welcome to the Projects page

Here i will be showcasing some of the projects i have completed through my journey into learning different programming languages. These can all be found on my Github. Click Here ! Or you can click on the images to take you straight to the file on Github.

The calculator App

This App allowed me to showcase different JavaScript concepts and kills. I used the Document Object Model(DOM) several times like "querySelectorAll" and "addEventListener" in order to interace with and manipulate HTML elements based on user input. I also utilised event handling to make the calculator interactive responding to different button clicks. Functions were also used to validate and prepare input, to show my ability to break down problems into managable pieces of code.

Stopwatch

This stopwatch project allowed me to work with timing functions like "setInterval" and "clearInterval" to manage the stopwatch's start, stop and reset functionalities . I also practiced basic arithmetic and string formatting to convert milliseconds to into a readable time format , ensuring a smooth user experience with zero padded numbers for consistent display.

Weather Dashboard

This weather dashboard project allowed me to practice working with asynchronous JavaScript using the "fetch" API to retrieve external data. I gained experience handling user input and events, to trigger data retrieval and update the interface dynamically. I also learned to manage and handle errors with "try-catch" blocks, ensuring a smooth user experience even when issues arise. Additionally, I practiced extracting and displaying specific data from a complex JSON object, reinforcing my understanding in dynamic content rendering in web applications.
